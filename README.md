# Suksham Gupta – Portfolio Website

This repository contains my personal portfolio website built to showcase my skills, projects, education, and professional experience as a Frontend Developer. The portfolio follows a clean, modern design inspired by professional developer portfolios and supports both light and dark themes for better readability.

Live Website:  
https://colorful-portfolio-i5a5.vercel.app/

---

## About the Project

This portfolio is designed as a professional web presence rather than a traditional resume download site. It highlights my technical background, hands-on project experience, and internships in a clear and structured manner.

The design focuses on minimalism, smooth animations, and proper spacing to create a professional impression for recruiters, hiring managers, and collaborators.

## Getting Started (Local Setup)

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/Suksham-123/colorful-portfolio.git

2. Navigate to the project directory
cd colorful-portfolio

3. Install dependencies
npm install

4.Start the development server
npm start

The application will run at http://localhost:3000.

Deployment

This project is deployed using Vercel.
Any changes pushed to the main branch automatically trigger a new deployment.
